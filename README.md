# Quantium Retail Analytics Job Simulation

> End-to-end retail analytics project focused on customer behavior analysis, trial store evaluation, and business reporting using Python.

---

## Project Overview

This repository contains my completed solution for the **Quantium Retail Analytics Job Simulation**, where I performed real-world retail data analysis to generate commercial insights and business recommendations.

The project follows the complete analytics workflow—from data preparation and exploratory analysis to trial store evaluation and executive reporting. The objective is to demonstrate how data can be transformed into actionable business decisions using analytical techniques and clear communication.

---

## Business Problem

A retail company wants to better understand customer purchasing behavior within the **Chips** category and evaluate whether a new in-store trial strategy successfully improved store performance.

The analysis aims to answer questions such as:

- Who are the most valuable customer segments?
- How do purchasing patterns differ across customer groups?
- Which stores should be selected as control stores for comparison?
- Did the in-store trial improve sales and customer growth?
- What recommendations can be provided to support future business decisions?

# Task Summary

## Task 1 — Customer & Category Analysis

Performed exploratory data analysis to understand customer purchasing behavior within the Chips category.

Main activities:

- Data cleaning and preprocessing
- Customer segmentation
- Sales trend analysis
- Life stage analysis
- Premium customer analysis
- Business insight generation

Deliverables:

- Jupyter Notebook
- Analysis Report
- Cleaned Dataset

---

## Task 2 — Trial Store Evaluation

Evaluated the effectiveness of three trial stores by selecting statistically similar control stores and comparing their performance.

Main activities:

- Monthly performance aggregation
- Correlation analysis
- Magnitude similarity analysis
- Control store selection
- Trial impact evaluation
- Business conclusion

Deliverables:

- Jupyter Notebook
- Analysis Report
- Processed Dataset
